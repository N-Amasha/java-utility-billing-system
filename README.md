# Java Utility Billing System

A console-based Java application designed to manage customer data and automate electricity bill calculations. This project demonstrates core Object-Oriented Programming (OOP) concepts.

## Key Features
- **Encapsulation:** Uses private fields with public Getters and Setters.
- **Constructors:** Implements parameterized constructors for clean object initialization.
- **Billing Logic:** Includes a tiered calculation method:
    - Up to 100 units: 20/unit
    - 101 to 300 units: 30/unit
    - Above 300 units: 40/unit
- **User Input Handling:** Integrated `Scanner` class with buffer management for seamless data entry.

## Setup & Usage
1. Clone this repository.
2. Open the project in **IntelliJ IDEA**.
3. Navigate to `src/Q4/CustomerApp.java`.
4. Run the file and follow the terminal prompts.
